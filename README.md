# VLESS Extractor

**VLESS Extractor** — это мощная утилита на Python с минималистичным графическим интерфейсом (GUI), созданная для удобного извлечения сырых VLESS-ссылок из URL-адресов подписок. Она специально разработана для обхода современных сетевых ограничений, систем глубокого анализа трафика (DPI) и защиты Cloudflare, которые часто блокируют стандартные запросы.

## ❓ Для чего это нужно?

Многие провайдеры VPN и прокси (например, на базе панелей Remnawave, 3x-ui или Marzban) выдают пользователям единую ссылку-подписку. 
Проблема в том, что:
1. **Нужны сырые ссылки:** Не все клиенты (особенно на роутерах) умеют работать с авто-подписками. Часто требуются именно "сырые" строки формата `vless://...`, чтобы импортировать нужный сервер вручную или поделиться им.
2. **Защита от парсинга:** Провайдеры часто защищают свои подписки (через Cloudflare) или отдают вместо ссылок красивую HTML-страничку, если открыть ссылку в обычном браузере. Вытащить из неё данные бывает непросто.
3. **Блокировки:** В регионах со строгой интернет-цензурой провайдеры (и системы вроде ТСПУ) блокируют сами домены подписок по TLS-отпечатку. Из-за этого клиенты просто не могут обновить список серверов.

Эта утилита решает все эти проблемы. Она пробивается через защиту, маскируется под нужный VPN-клиент, обходит блокировки на уровне TLS-рукопожатий и выдает вам готовые рабочие VLESS-ссылки в виде обычного текста.

## ✨ Особенности (Фишки)

- **Умное извлечение данных:** Автоматически декодирует подписки из формата Base64 и умеет вытаскивать ссылки даже из сложных HTML-страниц (например, панелей Remnawave).
- **Обход блокировок и анти-бан (DPI Bypass):** 
  - Программа автоматически притворяется разными VPN-клиентами (eVPN, v2rayN, Shadowrocket, NekoBox и др.), подставляя нужные `User-Agent`, чтобы гарантированно получить сырые ссылки от сервера.
  - Уникальный механизм изменения отпечатка SSL/TLS (Client Hello): позволяет обходить защиту от ботов Cloudflare и блокировки со стороны ТСПУ/провайдеров, из-за которых обычные скрипты выдают ошибку `ssl timeout`.
- **Надежность:** Встроена система авто-повторов (retries) при нестабильном соединении и обрывах связи.
- **Чистый интерфейс (GUI):** Удобное окно, написанное на `tkinter` (ttk) — просто вставьте ссылку и нажмите кнопку.
- **Генератор HWID:** Встроенная функция генерации и сохранения уникальных аппаратных идентификаторов (HWID) для панелей, требующих привязку к устройству.
- **Быстрый экспорт:** Копируйте все извлеченные ссылки в буфер обмена одним кликом или сохраняйте их прямо в `.txt` файл.

## 🚀 Установка и использование

1. **Установите Python:** Убедитесь, что у вас установлен Python версии 3.10 или новее.
2. **Скачайте проект:** Склонируйте репозиторий или просто скачайте файл `main.py`.
3. **Никаких сторонних библиотек:** Проект написан исключительно на встроенных библиотеках Python. Вам **не нужно** ничего устанавливать через `pip install`!
4. **Запустите программу:**
   ```bash
   python main.py
   ```

### Работа из консоли (CLI)
Вы также можете запустить скрипт напрямую из терминала без открытия графического окна:
```bash
python main.py "https://ваш-адрес-подписки.com" -o links.txt
```

## 📸 Как это работает
1. Запустите программу.
2. Вставьте URL вашей подписки в поле ввода.
3. *(Опционально)* Сгенерируйте новый HWID, если провайдер требует свежий идентификатор устройства.
4. Нажмите **Extract**. Скрипт сам подберет лучший профиль клиента, обойдет SSL-блокировки и вытащит сырые VLESS-ссылки.
5. Нажмите **Copy**, чтобы скопировать результат, или **Save to TXT**, чтобы сохранить ссылки в файл.
